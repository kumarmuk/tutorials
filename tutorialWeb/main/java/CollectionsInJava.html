SortedSet Interface

SortedSet interface is implemented by the TreeSet concrete class.
  • Elements are sorted in ascending order by default
  • Comparator can used to provide a sort order
  • Store unique values, nulls are not allowed
  • Slow on insertion and retrieval of data - O(log n) in time complexity
  • Fast for retrieving and accessing data
  • Other implementations of SortedSet are
    • TreeSet
    • ConcurrentSkipListSet

The following is an example of using a TreeSet with a Comparator 

   public static void main(String[] args) {
        
                        Comparator<Terminal> comparatorUpTime = new Comparator<Terminal>() {
                    
                          @Override
                          public int compare(Terminal t1, Terminal t2) {
                            if (t1.getUpTime() > t2.getUpTime()) {
                              return 1;
                            } else if (t1.getUpTime() == t2.getUpTime()) {
                              return 0;
                            } else {
                              return -1;
                            }
                          }
                        };
                    
                        SortedSet<Terminal> terminalSet = new TreeSet<Terminal>(comparatorUpTime);
                    
                        ValueGenerator vg = new ValueGeneratorImpl();
                    
                        for (int i = 1; i <= 1000; i++) {
                          Terminal t = new Terminal(vg.generateTerminalId(), vg.generateTerminalType(), vg.generateUpTime(),
                              vg.generateMonitorStatus(), vg.generateMonitoringFrequency());
                          System.out.println(t.toString());
                          terminalSet.add(t);
                        }
                    
                        terminalSet.forEach(terminal -> {
                          System.out.println(terminal.toString());
