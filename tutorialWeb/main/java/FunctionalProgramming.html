Functional - Programming in Java

Functional in nature means some code or function that will take the input and produce an output without altering the source

Streams

  • All classes and interfaces of the Stream API are contained in the java.util.stream package
  • API allows for creation of declarative pipelines for transformation of Arrays and Collections
  • Unlike Collections a stream does not need an iterator to go through all of its elements
    • Allow for operations to be lazy
    • Streams allow for sequential and parallel execution, filtering based on given predicates, mapping of objects, and much, much more
    • Work well wih lambda expressions
    • Stream are consumables, stream operations create streams, items once visited in a stream can not be revisited, rather a new stream will have to be created.
    • Streams can also be infinite
  • Streams are not collections and unable to store data, they can be termed as wrappers for Collections and Arrays

    • IntStream, LongStream, DoubleStream : they all extend the BaseStream
